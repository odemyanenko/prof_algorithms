1. Отсортируем коробки по убыванию размеров, начиная с самой большой.

2. Примем в начале, что грузовик пустой.

3. Поместим коробку с наибольшим размером, которая помещается в грузовик, в самом удобном для этого месте.

4. Удаляем эту коробку из списка доступных коробок.

5. Повторяем шаг 3,4 пока есть доступные коробки, которые могут поместиться в грузовик.

Полученное решение с помощью жадной стратегии не всегда будет оптимальным.
Можно не учесть оптимальное расположение коробок и может оставтся незаполненное пространство.
